1-To create repository 
git init

2-To link repo to github
git remote add origin "github link"

3- To pull files fro git hub
git pull origin master

4- To see the new changes in local repository
git status

5- To add the file/s to indexcommit
git add <file name> or git add -A (For all the files)

6- To do the commit in local repo
git commit -m "commit msg" or git commit -m "commit msg" for all files

7-To view logs
 git log
 
 8- To create a branch in master branch
 git branch <branch name>
 
 9- To copy the code from master branch or switch between branches
 git checkout <branch name>
